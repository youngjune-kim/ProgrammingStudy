# 2주차 과제

## 과제1

3개의 정수를 입력받아 최대값을 구하자.

### 입력

3개의 정수를 입력받는다.

#### example

```
첫번째 수: 123
두번째 수: 438
세번째 수: 307
```

### 출력

3개의 정수 중 최대값을 출력한다.

#### example

```
최대값은 438
```

## 과제 2

정수 배열을 선언하여 성적을 입력받은 후 평균값을 구하고, 평균 성적에 따라 A ~ F 까지의 학점을 부여하자.

90점 이상: A
80점 이상: B
70점 이상: C
60점 이상: D
60점 아래: F

### 입력

5개의 성적을 입력받는다.

#### example

```
국어: 60
수학: 98
영어: 90
사회: 70
과학: 85
```

### 출력

성적에 평균값에 따른 학점을 출력한다.

#### example

```
학점은 B
```

## 과제3

박물관 단체 입장 손님의 입장료 계산료 총합을 계산하자.

단체 손님은 10명이라고 가정한다.

10명의 나이를 각각 입력받은 후, 나이에 따른 요금을 합산하여 출력한다.

기본 요금: 10,000원
65세 이상: 25% 할인
19세 이하: 20% 할인
7세 이하: 50% 할인
3세 이하: 무료

### 입력

10명 손님의 각각의 나이를 입력받는다.

#### example

```
1번 손님: 70
2번 손님: 64
3번 손님: 30
4번 손님: 7
5번 손님: 5
6번 손님: 17
7번 손님: 29
8번 손님: 28
9번 손님: 65
10번 손님: 2
```

### 출력

10명 손님의 요금 총합을 출력한다.

#### example

```
단체 손님 요금 총합은 73000원
```
